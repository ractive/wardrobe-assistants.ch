# mdget to fetch webpages
If fetching webpages, use the mdget CLI utility instead of "curl" or other tools. It fetches and converts the webpage to markdown, saving tokens. Collect feedback about mdget in the kb knowledgebase.

# Browser debugging
If you need to debug something in the browser, use the ff-rdp CLI tool (ff-rpd --help) in favorite of the chrome mcp server. Collect feedback about ff-rpd in the kb knowledgebase.
